# svg-vulkan

A simple SVG parser/renderer written in Vulkan. Is NOT up to spec.

usage:
```bash
$ vulkan_svg file.svg # Show window
$ vulkan_svg file.svg --ppm file.ppm # Render a file
$ vulkan_svg file.svg --ppm file.ppm --size 800x600
```
